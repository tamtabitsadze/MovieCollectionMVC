# Movie Collection MVC

Aplikacja webowa stworzona przy użyciu ASP.NET Core MVC, Entity Framework Core oraz SQLite.

## Funkcjonalności

- Dodawanie filmów
- Edycja filmów
- Usuwanie filmów
- Wyświetlanie listy filmów
- Zapisywanie danych w bazie SQLite

## Technologie

- ASP.NET Core MVC
- Entity Framework Core
- SQLite
- HTML
- CSS

## Baza danych

Aplikacja wykorzystuje bazę danych SQLite do przechowywania informacji o filmach.

## Model Movie

Model zawiera następujące pola:

- Id
- Title
- Director
- Rating
